# GitHub Sponsors — 1-click setup

> Free passive revenue. 1% transaction fee (Stripe Connect). No monthly cost.
> Korean 사업자 OK. Receive in USD → wire to Korean bank.

## Step 1 — Enable Sponsors (sign in needed, ~3 min)

Go to: https://github.com/sponsors

Click **"Join the waitlist"** or **"Set up GitHub Sponsors"**. They'll ask:
1. Whether you're an individual or an organization → **Individual**
2. Email verification (you've done this)
3. Stripe Connect onboarding (KYC: 사업자등록증 + 통장 사본 + 신분증 사진)

Approval usually takes 1-3 days.

## Step 2 — Paste this into your sponsorship goals + bio

> Building ClonePilot — YouTube → live MVP in 2 minutes via MCP.

## Step 3 — Add the badge to README

Once approved, your sponsor URL becomes `https://github.com/sponsors/a01050398694-commits`. Add this to the top of README.md:

```markdown
[![Sponsor on GitHub](https://img.shields.io/github/sponsors/a01050398694-commits?label=Sponsor&logo=github&style=flat-square)](https://github.com/sponsors/a01050398694-commits)
```

## Step 4 — Add `.github/FUNDING.yml` (already prepped)

This file already exists in `.github/FUNDING.yml` — once Sponsors is enabled, GitHub auto-shows a "Sponsor" button at the top of the repo page.
